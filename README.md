# Dashboard CSS3 — Camilo Páez

## 📌 Descripción

Este proyecto consiste en la implementación de un dashboard web responsivo utilizando CSS Grid para la estructura principal y Flexbox para los componentes internos, sin el uso de frameworks externos.

## 🚀 Tecnologías utilizadas

- HTML5
- CSS3 (Grid y Flexbox)

## 📂 Estructura del proyecto

- index.html
- css/dashboard.css
- pantalla-grande.png
- pantalla-pequena.png

## ▶️ Cómo ejecutar el proyecto

1. Abrir la carpeta en Visual Studio Code
2. Abrir el archivo `index.html`
3. Ejecutar con la extensión **Live Server**
